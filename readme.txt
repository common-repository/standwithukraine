Inform visitors of your website that you support Ukraine. Tell them how they can help too - donate to Ukrainian Army and government.

== Description ==

This plugin is written and distriubted only in infromation purposes. No payments and no analytics are implemented through this plugin.

This plugin simply adds button to the bottom of the page or a full-page line (at the top or at the bottom of the page). See all the available styles on the screenshots below. When user clicks the button - a popup appears with the following text:

#StandWithUkraine

We don't know how long the war will last. But what we do know is that we can't stand aside and watch.

The fastest way you can help too is to support Ukraine financially. The National Bank of Ukraine (NBU) has opened a multi-currency account for that purpose.

This account accepts donations in US, Canadian and Australian dollars, euros, British pounds, Swiss francs, yuan and yen.

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Stand with Ukraine settings page in menu
4. Select format and style for the banner.
